# 区别

C++是编译型语言：首先将源代码编译生成机器码，再有机器运行机器码。执行速度快，效率高，依赖编译器，跨平台性较差

Java是解释型语言：源代码不是直接翻译成机器语言，而是先翻译成中间代码，再由解释器对中间代码进行解释运行。执行速度慢，效率低，依赖解释器，跨平台性好。
